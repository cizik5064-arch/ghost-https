# ghost-https
Сделанный с помощью ИИ, корректированный человеком, новый http/https прокси сервер, основанный на .jar формате

# Сборка
mvn clean package

# Зависимости
Java 21

# Настройка
Не рекомендую вам скачивать релиз, советую скачать исходный код
и собрать но уже со своими настройками, так как
в данном релизе настройки в исходном коде.

В Main.java (класс Main) вы можете найти:
private static final int PORT = 26288;
    private static final String BIND_ADDRESS = "0.0.0.0";
    private static final String PROXY_USER = "changeme";
    private static final String PROXY_PASS = "changeme";
это и есть базовые настройки которые вы можете поменять,
порт, также креденшиалы, и адрес который слушает прокси.

после того как вы изменили свои настройки, вам нужно установить maven,
в яндексе сами разберетесь, после установки перезапускаем компьютер
и открывает Командную строку (через WIN + R: cmd), переходим в корневой
каталог проекта (cd путь), и оттуда пишем команду из сборки, эта команда
скачает все нужные библиотеки и скомпилирует приложение уже с вашими настройками.

Скомплированное приложение вы можете найти в target/server.jar.

# Запуск
Переходим в нашу VDS, и загружаем туда наше приложение. дальше запускаем его, и все.
прокси работает по тому адресу и порту (с теми же креденшиалами) которые вы указали
в пункте "Настройка"
